# General setter function for Risoe.BINfileData objects

The function provides a generalised access point for specific
[Risoe.BINfileData](https://r-lum.github.io/Luminescence/reference/Risoe.BINfileData-class.md)
objects. Depending on the input object, the corresponding function will
be selected. Allowed arguments can be found in the documentations of the
corresponding
[Risoe.BINfileData](https://r-lum.github.io/Luminescence/reference/Risoe.BINfileData-class.md)
class.

## Usage

``` r
set_Risoe.BINfileData(
  METADATA = data.frame(),
  DATA = list(),
  .RESERVED = list()
)
```

## Arguments

- METADATA:

  x

- DATA:

  x

- .RESERVED:

  x

## Value

A
[Risoe.BINfileData](https://r-lum.github.io/Luminescence/reference/Risoe.BINfileData-class.md)
object.
