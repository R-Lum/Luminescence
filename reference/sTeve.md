# sTeve - sophisticated tool for efficient data validation and evaluation

This function provides a sophisticated routine for comprehensive
luminescence dating data analysis.

## Usage

``` r
sTeve(n_frames = 10, t_animation = 2, n.tree = 7, type = NULL)
```

## Arguments

- n_frames:

  [integer](https://rdrr.io/r/base/integer.html) (*with default*): n
  frames

- t_animation:

  [integer](https://rdrr.io/r/base/integer.html) (*with default*): t
  animation

- n.tree:

  [integer](https://rdrr.io/r/base/integer.html) (*with default*): how
  many trees do you want to cut?

- type:

  [integer](https://rdrr.io/r/base/integer.html) (*optional*): Make a
  decision: 1, 2 or 3

## Value

Validates your data.

## Details

This amazing sophisticated function validates your data seriously.
