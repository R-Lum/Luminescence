# Example data for scale_GammaDose()

An example data set for the function
[`scale_GammaDose()`](https://r-lum.github.io/Luminescence/reference/scale_GammaDose.md)
containing layer specific information to scale the gamma dose rate
considering variations in soil radioactivity.

## Format

A [`data.frame`](https://rdrr.io/r/base/data.frame.html). Please see
`?scale_GammaDose()` for a detailed description of its structure.

## Version

0.1

## Examples

``` r
## Load data
data("ExampleData.ScaleGammaDose", envir = environment())
```
